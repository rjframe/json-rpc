    Function RPCClient.batch

    Execute a batch of function calls.

    RPCResponse[] batch (
      Tuple!(string,JSONValue)[] methods
    );

    Parameters

    NameDescription
    methods An array of Tuple of values (method-name, JSONValue(parameters))

    Returns

    An array of RPCResponse objects, in the same order as the respective request.

    Notes

    This does not yet support notifications in batch requests.

    Example

    import std.typecons : tuple;
interface API {
    void func1(int a);
    long func2(string s);
    long func3();
}
auto client = RPCClient!API("localhost", 54321);

auto responses = client.batch([
        tuple("func1", JSONValue(50)),
        tuple("func1", JSONValue(-1)),
        tuple("func2", JSONValue("hello")),
        tuple("func3", JSONValue()),
        tuple("func1", JSONValue(123))
]);
