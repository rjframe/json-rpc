    Class RPCServer

    Implementation of a JSON-RPC server.

    class RPCServer(API, Transport)
      
    if (is(API == class) && is(Transport == struct));

    This implementation only supports communication via TCP sockets.

    Constructors

    NameDescription
    this Construct an RPCServer!API object to communicate via TCP sockets.
    this Construct an RPCServer!API object to communicate via TCP sockets.
    this Construct an RPCServer!API object.

    Methods

    NameDescription
    listen Listen for and respond to connections.

    Compile Time Parameters

    API = A class or struct containing the functions available for the client to call.
    Transport = The network transport to use for data transmission. By default, a TCPTransport.

    Example

    class MyFunctions {
    long func(string param) { return 56789; }
}

// Bind to a local port and serve func on a platter.
auto serve = new RPCServer!MyFunctions("127.0.0.1", 54321);
serve.listen();
