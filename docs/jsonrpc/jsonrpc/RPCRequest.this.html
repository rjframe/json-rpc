    Function RPCRequest.this

    Construct an RPCRequest with the specified remote method name and arguments.

    this(T) (
      T id,
      string method,
      JSONValue params = JSONValue()
    )
    if (isNumeric!T || isSomeString!T || is(T : typeof(null)));

    The ID must be a string, a number, or null; null is not recommended for use as an ID.

    Parameters

    NameDescription
    id The ID of the request.
    method The name of the remote method to call.
    params A JSON string containing the method arguments as a JSON Object or array.

    Throws

    InvalidArgumentException if the json string is not a JSON Object or array.
