    Function TCPTransport.listen

    Listen for client requests.

    void listen(alias handler) (
      API api,

    listen will call the specified handler function in a new thread to handle each client it accepts.

    NameDescription
    api An instantiated class with the methods to execute.
    host The hostname of the server on which to listen.
    port The port on which to listen.
